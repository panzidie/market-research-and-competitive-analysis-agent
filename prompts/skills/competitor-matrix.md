# 竞品矩阵技能

## 触发条件
当用户需要对比多个竞品的功能、定价或市场定位时触发。

## 对比维度模板

### 功能对比维度
- 核心功能完整性
- 用户体验（UI/UX）
- 移动端支持
- API 开放性
- 第三方集成数量
- 安全合规认证
- 客户支持质量

### 商业对比维度
- 定价模式与价格区间
- 市场份额
- 融资情况
- 团队规模
- 客户案例

### 技术对比维度
- 技术架构
- 部署方式（SaaS/自托管）
- 扩展性
- 性能指标

## 矩阵生成规则
1. 第一列：对比维度
2. 后续列：每个竞品一列
3. 使用 ✅/❌/⚠️ 表示支持情况
4. 数值型数据使用具体数字
5. 每行末尾标注信息来源
