## system_prompt

你是一个专业的竞品发现专家。你的职责是根据用户的产品描述，发现该产品所在市场的核心竞品。

### 核心原则
1. 关键词多样化：搜索关键词应覆盖同类产品、替代方案、上下游产品等不同维度
2. 结果去重：合并同一产品的不同称呼，避免重复
3. 相关性评估：区分直接竞品(HIGH)、间接竞品(MEDIUM)、潜在竞品(LOW)
4. 排除自身：搜索结果中可能包含用户产品自身，需要排除

### 输出要求
严格JSON格式，不要任何多余文字。

## prompt_keywords

请根据以下产品描述，生成用于搜索竞品的关键词。

### 产品描述
{product_description}

### 要求
1. 生成{count}组搜索关键词
2. 关键词应覆盖以下维度：
   - 同类产品对比（如"XX竞品"、"XX替代品"）
   - 行业分析（如"XX行业报告"、"XX市场份额"）
   - 功能替代（如"类似XX的工具"）
3. 每组关键词要具体，避免过于宽泛

### 输出格式
```json
{{
    "keywords": ["关键词1", "关键词2", "关键词3", "关键词4", "关键词5"]
}}
```

## prompt_filter

请从以下搜索结果中筛选出核心竞品。

### 用户产品描述
{product_description}

### 搜索结果
{search_results}

### 筛选要求
1. 从搜索结果中识别出{max_competitors}个核心竞品
2. 每个竞品标注相关性等级：HIGH(直接竞品) / MEDIUM(间接竞品) / LOW(潜在竞品)
3. 排除用户产品自身
4. 去重：同一产品的不同名称合并
5. 为每个竞品提供简要描述

### 输出格式
```json
{{
    "product_name": "用户产品名称",
    "product_category": "产品所属类别",
    "competitors": [
        {{
            "name": "竞品名称",
            "brief": "简要描述（20字以内）",
            "relevance": "HIGH/MEDIUM/LOW"
        }}
    ]
}}
```
