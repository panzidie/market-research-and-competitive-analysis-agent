# SWOT 分析技能

## 触发条件
当用户需要对特定竞品或整体市场进行 SWOT 分析时触发。

## 分析框架

### 优势 (Strengths)
- 产品差异化特点
- 技术壁垒
- 品牌影响力
- 客户忠诚度
- 成本优势

### 劣势 (Weaknesses)
- 功能缺失
- 用户体验问题
- 市场覆盖不足
- 资源限制

### 机会 (Opportunities)
- 市场增长空间
- 新技术应用
- 政策利好
- 竞争对手弱点

### 威胁 (Threats)
- 新进入者
- 替代品
- 政策风险
- 技术变革

## 输出格式
每个维度至少列出 3-5 点，每点附带简要说明和来源。
